# AliLegacy

**AliLegacy** is a small open-source technical experiment documenting  
a symbolic Solidity contract and a full development workflow  
in a constrained environment (**Android / Termux / ARM64**).

---

## What this project is

- A personal open-source technical milestone
- A symbolic Solidity smart contract (`AliLegacy`)
- An experiment in **technical sovereignty**, minimal tooling, and accessibility
- Built and maintained in a non-traditional environment (mobile + Termux)

---

## Current status

- Solidity contract implemented
- Repository publicly available
- Testing tools (Forge) planned — pending environment setup on ARM64

---

## Motivation

This project exists to document what is possible when modern Web3
development is approached with limited resources and full transparency.

# farcaster-mini

Minimal **Next.js 15** project for building **Farcaster Frames** and **mini apps**, developed using **Termux on Android**.

هذا المشروع مهيأ ليكون نقطة انطلاق خفيفة وسريعة لتطوير تطبيقات Farcaster باستخدام أحدث إصدار من Next.js.

---

## ✨ Features

- ⚡ Next.js 15 (App Router)
- 🧩 Farcaster Frame SDK
- 📱 يعمل بالكامل على Termux (Android)
- 🧪 جاهز للتكامل مع GitHub Actions
- 🔒 إعدادات نظيفة مع `.gitignore` محسّن

---

## 📦 Tech Stack

- **Framework:** Next.js 15
- **Language:** TypeScript
- **UI:** React 18
- **SDK:** `@farcaster/frame-sdk`
- **Runtime:** Node.js 18 / 20
